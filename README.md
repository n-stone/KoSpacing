# KoSpacing

## 1. Introduce

- [kospacing](https://github.com/haven-jeon/KoSpacing)은 전희원 연구원님께서 한글 자동 띄어쓰기를 쉽게 활용할 수 있도록 만든 패키지 입니다.  
- 띄어쓰기는 형태소 분석 이전에 반드시 수행해야 되는 중요한 과정이다. 
<br></br>
